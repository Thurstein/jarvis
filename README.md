# Jarvis

Jarvis es un asistente de escritorio inspirado en J.A.R.V.I.S. de Iron Man, desarrollado en Python y ejecutado localmente mediante modelos de lenguaje de Ollama.

## Características actuales

* Reconocimiento de voz.
* Síntesis de voz (TTS).
* Sistema de herramientas.
* Apertura de programas.
* Apertura de archivos y carpetas.
* Control multimedia.
* Control de volumen.
* Memoria persistente entre sesiones.
* Búsqueda automática de recuerdos.
* Personalidad basada en J.A.R.V.I.S.

## Tecnologías

* Python
* Ollama
* Qwen
* Whisper
* Piper TTS

## Estado del proyecto

Proyecto en desarrollo activo.

Actualmente Jarvis puede mantener conversaciones, ejecutar herramientas locales y recordar información importante entre sesiones.
